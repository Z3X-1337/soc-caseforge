# Roadmap

## v0.2.0 — stronger evidence model

- ISO-8601 timestamp normalization with explicit timezone handling.
- CSV and generic JSON event adapters.
- Case notes, dispositions, and analyst decisions.
- Configurable suppression and allowlist policies.
- Exportable evidence manifest with file hashes.

## v0.3.0 — analyst workflow

- Local browser interface.
- Pluggable parser and detection registries.
- STIX 2.1 indicator export.
- SARIF and HTML reports.
- Detection test datasets and false-positive guidance.

## Product boundary

SOC CaseForge will remain analyst-assistance software. It will not claim autonomous investigation, attribution, exploitability, or automatic containment without human authorization.
